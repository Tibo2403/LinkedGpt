# Portfolio Brief

## Problem

Professionals need a focused workspace to draft LinkedIn content, plan outreach, and review engagement without wiring real integrations during a demo.

## What This Shows

- React and TypeScript product UI.
- Demo-safe architecture with no required production secrets.
- Content generation, scheduling, activity tracking, and analytics workflows.
- CI-backed frontend quality checks.

## Demo Flow

1. Start the app in demo mode.
2. Generate a LinkedIn post from a short topic.
3. Review suggested outreach messages.
4. Open analytics and calendar views.
5. Show that demo mode works without Supabase, OpenAI, LinkedIn, or calendar credentials.

## Recruiter Notes

This project is useful to demonstrate product thinking, frontend architecture, environment hygiene, and safe integration boundaries.
